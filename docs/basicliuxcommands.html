Basic Linux Commands

Log into the hub.docker.com/ and create the repository. After creating repository with description click create.

Log into the Docker Hub from the command line and just with your own username and email that you used for the account. Enter your password if everything worked you will get a message. “login Succeeded”

Check the image ID using docker images

You will see repository, tag, Image Id, created and size.

Type up docker tag bb38976d03cf yourhubusername/reponame:firsttry

We will push your image to the repository that you have created by docker push yourhubusername/reponame

Then your image is now available for everyone to use. To save a Docker image after you have pulled, committed, or built it you use the docker save command. For example, lets save a local copy of the reponame docker image we made

docker save reponame > reponame.tar

After you want to load that docker container from the archived tar file in the future, we can use the docker load command:docker load --input reponame.tar

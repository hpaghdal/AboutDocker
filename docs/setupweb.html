Docker Information:

Docker is a software technology to create, deploy and run an application or group of applications in containers. A container is a package that contains everything needed to run your application. That includes the operating system and any programs on that operating system that are needed to run the application.

Docker is extremely useful for deploying scaling services to production. You can launch new instances which only have to pull your Docker image down and run the entry command to get a running application. It is also useful for developing new applications.
